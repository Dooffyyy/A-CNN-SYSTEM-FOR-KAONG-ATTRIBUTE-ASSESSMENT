Project Overview

KRIPEN (Kaong Ripeness Evaluation Network) is an AI-based web system designed to assess the ripeness of kaong fruits through image analysis. By using a Convolutional Neural Network (CNN), KRIPEN automatically classifies fruits as Ripe, Unripe, or Rotten, helping farmers and producers determine the ideal harvest time and improve product quality.

Our system revolutionizes traditional fruit assessment methods by providing instant, accurate, and consistent ripeness evaluation through advanced machine learning algorithms.

Technology Behind KRIPEN

KRIPEN uses a Convolutional Neural Network (CNN) architecture trained on images of kaong fruits at various stages of ripeness. The model was trained using annotated datasets and optimized for accuracy using transfer learning and image augmentation techniques.

The web system was developed using modern web technologies, integrating the trained CNN model to provide real-time predictions through a user-friendly interface.
